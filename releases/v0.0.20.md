# v0.0.20

> Released: 2026-04-04

## Highlights

- **Fix: Unicode surrogate crash** — Heartbeat runs no longer fail with "invalid input syntax for type json" when agent output contains partial emoji surrogates. Added `sanitizeForJsonb` utility that cleans all JSON data before PostgreSQL insertion.
- **README redesign** — Demo video, problem/solution table, two-column feature grid, killer differentiators (CAG, RAG, token optimization, process security).
- **Operations guide** — New `OPERATIONS.md` with production/dev workflow, launchd management commands, release process, and emergency procedures.

## Fixes

- **PostgresError on heartbeat context** — LLM output with broken Unicode surrogates (`\uD800-\uDFFF`) caused `INSERT` failures on `heartbeat_runs.context_snapshot`. Now sanitized at all insertion points with `sanitizeForJsonb()`.

## Improvements

- **Dev version clarity** — Repo `package.json` versions changed from misleading `0.0.3` to `0.0.0-dev`. Production installs show the real version; dev always shows `v0.0.0-dev`.
