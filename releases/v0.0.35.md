# v0.0.35

> Released: 2026-04-17
> Opus 4.7 beta-feature passthrough in `claude-local` + landing FAQ update.

## What's new

The `claude_local` adapter now forwards the Claude Code CLI 2.1.101+
flags that unlock Anthropic's new Opus 4.7 beta features. No new
adapter needed — it all happens through the existing adapter, so every
Claude agent in your FideliOS company can opt in per-agent.

### New agent-config fields (all optional)

| Field | Maps to CLI flag | What it does |
|-------|-----------------|--------------|
| `effort` | `--effort low/medium/high/max` | Trade depth-of-reasoning for cost/speed per agent. `low` for throwaway subagents, `max` for frontier problems. GA on Opus 4.7, Opus 4.6, Sonnet 4.6. |
| `fallbackModel` | `--fallback-model` | Auto-switch model when the primary is overloaded (e.g. primary Opus 4.7 → fallback Sonnet 4.6). |
| `maxBudgetUsd` | `--max-budget-usd` | Hard USD spend cap per run. The CLI aborts when reached. Complements FideliOS's per-agent monthly budgets. |
| `betas` | `--betas` | Comma/space-separated Anthropic beta-header names. See below. API-key users only. |

### Beta headers worth setting

- **`advisor-tool-2026-03-01`** — pair a cheaper Sonnet / Haiku executor with
  Opus 4.7 as a mid-generation advisor. Typical saving: 30-50% of
  Opus-solo cost at near-Opus quality on long agentic tasks.
- **`task-budgets-2026-03-13`** — Opus 4.7 sees a token countdown and
  self-paces the agentic loop, finishing gracefully rather than cutting
  off mid-action.
- **`fast-mode-2026-02-01`** — 2.5x output speed on Opus 4.6 at 6x token
  price (waitlist-gated).

Example agent config:

```yaml
model: claude-sonnet-4-6
effort: high
fallbackModel: claude-haiku-4-5-20251001
maxBudgetUsd: 5.00
betas: "advisor-tool-2026-03-01"
```
