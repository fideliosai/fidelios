# v0.0.14

> Released: 2026-04-02

## Fixes

- **Agent instructions safety** — HEARTBEAT.md template and all 25 deployed agent instructions patched: removed hardcoded `YYYY-MM-DD.md` file paths that caused "File not found" crashes. Agents now use `ls` to discover existing memory files before reading.
- **Pre-compiled context hint** — All 25 agents now have the "## Issue Context (pre-compiled by FideliOS)" hint in HEARTBEAT.md, so they use the pre-compiled context bundle instead of making redundant API calls.
- **op-secrets rule 3 expanded** — SA tokens must never be stored in databases, config files, `adapterConfig.env`, or any persistent storage. Only shell environment inheritance from `~/.zshrc`.
- **Session compaction test** — Updated test expectations to match balanced thresholds (50 runs, 24h).
