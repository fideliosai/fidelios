# v0.0.22

> Released: 2026-04-06

## Fixes

- **CJS/ESM crash on startup (Node.js v24)** — Fixed `fidelios run` crashing immediately with `require() cannot be used on an ESM graph with top-level await`. Root cause: `jsdom@28` pulls in `cssstyle@6` which tries to `require()` `@asamuzakjp/css-color@5.x`, an ESM-only module. Fixed by downgrading `jsdom` to `^27.0.0` in `@fideliosai/server`, which uses `cssstyle@5` and the CJS-compatible `@asamuzakjp/css-color@4.x`.
